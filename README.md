# 🎵 WaveTune – “Feel the Rhythm. Hear the Soul.”

WaveTune is a dynamic music and podcast web application built with HTML, CSS, JavaScript, PHP, and MySQL. It allows users to stream music and podcasts while giving admins the ability to manage content dynamically through a secure dashboard.

## 🚀 Features

- Stream music and podcasts directly in-browser  
- Responsive and modern UI with smooth transitions  
- Search and filter by genres or categories  
- Embedded audio player with full playback controls  
- Admin panel for managing songs, genres, and uploads  
- Database-driven content management system  

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL
